# 🛒 E-Commerce Sales Performance & Customer Insights

## 📌 Overview
End-to-end data analysis project on 100K+ real Brazilian e-commerce orders.
Includes revenue analysis, customer segmentation and delivery performance.

## 📊 Key Insights
- Monthly revenue grew from R$61K (Oct 2016) to R$1.5M (Nov 2017)
- São Paulo dominates with 18,163 orders — 2x more than Rio de Janeiro
- Credit card is the most preferred payment method (74% of orders)
- 93,349 customers segmented into VIP, Loyal, Potential, At Risk and Lost
- Average delivery time is 12 days — 105,033 orders delivered early

## 🛠️ Tech Stack
- Python, Pandas, Matplotlib
- Jupyter Notebook

## 📁 Dataset
Brazilian E-Commerce Public Dataset by Olist (Kaggle)
